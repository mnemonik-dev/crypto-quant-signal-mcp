Frequently Asked Questions

What is AlgoVault?
AlgoVault is an MCP server that provides AI trading agents with composite BUY/SELL/HOLD trade calls, cross-venue funding rate arbitrage scanning, and market regime detection across 5 exchanges — Hyperliquid, Binance, Bybit, OKX, and Bitget. It works with any MCP-compatible client including Claude, Cursor, and custom agents.
How does the trade call scoring work?
The scoring engine combines multiple orthogonal factors — momentum, trend structure, derivatives positioning, open interest dynamics, and volume conviction — into a single weighted composite score. Trade calls pass through regime-aware filters and adaptive post-processing gates before a final verdict is emitted. Only high-conviction trade calls are generated. Each trade call includes a confidence percentage and human-readable reasoning.
What exchanges does AlgoVault support?
AlgoVault generates full trade calls on 5 exchanges: Hyperliquid, Binance, Bybit, OKX, and Bitget. Each venue uses native candle, open interest, funding rate, and volume data. TradFi perpetuals (stocks, commodities, indices, FX) are available on Hyperliquid. Cross-venue funding arbitrage scanning covers HL, Binance, and Bybit. All data comes from public APIs — no exchange account or API keys required.
Is AlgoVault free to use?
Yes. The free tier includes BTC and ETH trade calls on 15m and 1h timeframes, top 5 funding arbitrage opportunities, and market regime detection for all assets — with no signup required. For all crypto and TradFi assets across all 11 timeframes, upgrade to Pro ($49/month) or pay per call via x402 micropayments (USDC on Base, starting at $0.01/call).
What is MCP (Model Context Protocol)?
MCP is an open standard by Anthropic that lets AI assistants connect to external tools and data sources. AlgoVault implements MCP so any compatible client (Claude, Cursor, VS Code, custom agents) can discover and use its trading tools automatically. Learn more at modelcontextprotocol.io.
